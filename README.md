RNA-seq Analysis of Chlorella vulgaris
Overview:

This project focuses on analyzing gene expression changes in Chlorella vulgaris under different environmental conditions.

Objectives:

Identify differentially expressed genes (DEGs)

Understand stress adaptation mechanisms

Perform pathway analysis

Tools Used:

DESeq2

KEGG

BLAST

Methodology:

RNA-seq data interpretation

Identification of gene expression changes

Functional pathway analysis

Results:

Identified genes involved in stress response

Observed metabolic pathway changes under different conditions

Skills Demonstrated:

RNA-seq analysis

Bioinformatics

Data interpretation
